# Hospital Power BI Dashboard

## Objective
To analyze patient visits, disease patterns, and pharmacy performance to support hospital decision-making.

## Tools Used
- Power BI Desktop
- Excel
- GitHub

## Dashboard Features
- KPI cards (Total Visits, Pharmacy Revenue, Avg Length of Stay)
- Disease trends over time
- Pharmacy cost breakdown by drug category
- County and department comparisons
- Interactive slicers for County, Diagnosis and Dates of Visit

## How to Use
Download the `.pbix` file and open it using Power BI Desktop.
